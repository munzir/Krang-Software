This is the ftread implementation using only somatic, and not comedi. 2012-12-10. Can Erdogan.

This code has three executables:

1- ftread01_print: Prints the force/torque readings to the command line.
2- ftread02_plot2d: Creates two 2D plots for the force and torque readings.
3- ftread03_plot3d: Creates two 3D plots for the force and torque readings.
