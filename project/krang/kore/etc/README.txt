This folder contains the configuration files for the Krang model in dart. The purpose of each 
subfolder is as follows:

- model: Contains the Krang.urdf and KrangBlocks.urdf. Write a world model and include these.
				 See the 01-World-Example.urdf that can be opened with grip-run. Will be installed.
- objects: 	Contains the object file defintions such as tables, pointers and etc. Will be 
						installed.
- extra: Contains the urdf files that were used to create the model. Will not be installed.
